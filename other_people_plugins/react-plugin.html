            • react-vis: Uber做的一款基于React.js的图表插件

            • react-chartjs: 混合 React 和 chart.js 的图表

            # 输入

            • Draft.js: 一个React 文本编辑器

            • React Select

            • React Lite Dropdown: 基于 React 的下拉列表插件

            • react-times: 一款时间选择插件，可以选择24或12小时制，以及黑白主题

            • react-dates: Airbnb开源的一款日期选择器

            • cleave.js

            • text-mask

            • Inputmask

            格式化 input 的输入

            • react-medium-editor

            开源编辑器 medium-editor 的 React 版本

            React Modal 的辅助神器
