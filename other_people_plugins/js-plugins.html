                          # JS PLUGIN

                          mark 一个 issue 地址

                          # 动画

                          • mo.js: 牛逼的动效库

                          • Bounce.js: 如其名，带有各种弹性效果的动画库，还挺有意思的

                          • anime: Javascript Animation Engine

                          • Move.js: 一个 JS 动效库。还没仔细看它文档

                          • Parallax.js: 一个做视差滚动的 js 库

                          • PROGRESS BAR.JS

                          • SpinThatShit

                          各种样式的 loading

                          • choreographer-js: A simple library to take care of complex CSS animations.

                          • ajaxload

                          • loadingapng

                          • loadinggif

                          • typed.js: 模拟打字效果，需要 jQuery 依赖

                          • ityped: 同样是模拟打字效果， 不需要 jQuery 依赖

                          # 页面 PPT

                          • reveal.js: 全屏切换。做 web ppt 的好东西

                          • Swiper: 专注触屏滑动，体会 Android 上 ViewPager 的爽快

                          • slick: 各种左右滑动效果

                          • fullPage.js: 上下全屏滑动。现在很多网站首屏上常见到的动效

                          • Headroom.js: 下滑页面隐藏导航栏，上滑展现导航栏

                          • iScroll: 处理页面滚动。滚动，缩放，平移，无限滚动，视差滚动，旋转功能

                          • scrollreveal: 滚动如丝般顺滑

                          • waypoints: Waypoints is the easiest way to trigger a function when you scroll to an element.

                          • glide: 轻量级的简单的 slider，用起来还行但是文档比较坑

                          # 媒体

                          • bLazy.js: 延迟图片加载，只有当图片首次出现在屏幕内的时候才进行加载。以此提高页面初始化速度

                          • Vidage: 全屏背景视频&图片

                          • Spectrum: 基于 jQuery 的 colorPicker，简单易用

                          • VideoJs: 用于视频处理的 js 库

                          • PhotoSwipe: 图片处理的库, 点击图片全屏展示，可左右切换图片

                          • Fabric.js: html5 canvas 绘图插件。挺强大的，基本上只有你想不到没有它做不到。提供的接口可以让你自己进行各种扩展

                          • Cropper: 基于 jQuery 的图片剪裁库

                          • Filterizr: Filterizr is a jQuery plugin that searches, sorts, shuffles and applies stunning filters over responsive galleries using CSS3 transitions.

                          • granim.js: 渐变效果

                          # 数据处理

                          • Cal-heatmap

                          是否还记得 github/gitlab 上个人中心里记录自己每日 push 贡献次数的日历表格？

                          • sigmajs

                          • DataTables: 一个基于 jQuery 的表格插件，可以实现各种表格排序功能

                          • tablesorter: 基于 jQuery 的表格插件，使用起来比 DataTables 简单非常多

                          • List.js: 给 <ul> 或 <table> 做数据排序/筛选的 js 插件

                          • Moment.js: 轻量级的快速处理时间的插件，便捷的输出各种格式的时间

                          # 前端缓存

                          • Basil.js: 集本地存储、Cookie 和 Session Storage API 于一体，使得前端的存储管理更加方便，允许完全的本地存储

                          • storage: Asynchronous browser storage with multiple back-ends (IndexedDB, WebSQL, localStorage)

                          • Store.js: 适用于所有浏览器的本地存储，不使用 cookies 或者 flash。会根据浏览器的不同选择 localStorage, globalStorage, 和 userData 作为存储引擎。

                          • js-cookie

                          # HTML 转换

                          • jsPDF: Render HTML to PDF，但是只有当 HTML 结构与 CSS 样式比较简单时体验较好，适用用把 markdown 类的文章转换为 PDF

                          • dom-to-image: 把 DOM 转换为图片。同样的，当 DOM 数量多或复杂的时候体验不好（渲染速度慢）

                          • to-markdown: html 转 markdown

                          # 写作

                          • snarkdown: 轻量级的 Markdown -> HTML 解析

                          • tinymce: 文本编辑器插件

                          • Caret.js和At.js :可用于控制 textarea 光标位置和数据插入，比较方便

                          • clipboard.js

                          • MediumEditor: 轻量级文本编辑器，没有通常的 TextArea 式的 UI，而会随着用户选择的文字而出现

                          • Quill: rich text editor

                          • Medium.js: 感觉挺新颖的，可以研究下

                          • SimpleMDE: 支持 markdown 的富文本编辑器

                          • marginotes: jQuery 插件，文章旁注的效果

                          • cleave.js

                          • text-mask: 格式化 input 的输入

                          • pell

                          # markdown 解析

                          • marked: 老牌 markdown 解析器

                          • marky-markdown: npm 官方的 markdown 解析插件

                          • Mditor(不推荐): 轻量级的 Markdown 预览插件

                          • editor: 还不错的 markdown 编辑器

                          • markdown-it: Markdown parser done right. Fast and easy to extend.

                          • pickadate.js: The mobile-friendly, responsive, and lightweight

                          • datedropper: 针对移动端的日期选择

                          • react-times: 基于 React 的时间选择组件，没有 jQuery 依赖

                          # 其他工具

                          • UNDERSCORE.JS: 工具类函数库，非常之方便。。但是有一点 --> 用太多会被它惯坏啊喂

                            推荐看它源码进行学习，深入理解那些函数底层的实现

                          另有汉化版文档

                          • favico.js: 在你网站的 favico 上展现小徽标，逼死强迫症

                          • Messenger: 一个 Message 弹窗组件，基于 jQuery 和 Backbone

                          • Modaal: JS 弹窗插件

                          • biu: biu 的一下冒出来一个光头~啊不，是弹窗

                          • Tipso: 鼠标 hover 上去以后弹出的一个提示框效果

                          • FlipClock.js: 类似日历翻页效果的插件

                          • PLEASE.JS: 生成随机色的 js 插件，带有过度的动画效果

                          • Feature.js: 快速有效的检测用户浏览器对各个 API 的兼容性

                          • Hilo: 一个跨终端的互动小游戏解决方案

                          • jQuery WeUI: WeUI 是微信官方团队针对微信提供的一个 H5 UI 库，它只提供了一组 CSS 组件。

                          • hammerjs: 手势解析库

                          • push.js: 浏览器通知插件

                          • Slidebars: jQuery 插件，创建侧边栏

                          • Leaflet: 开源地图组件

                          • fusejs: 无依赖 JavaScript 库，提供模糊搜索（近似字符串匹配）
